## 📝 О проекте

**Dananz** — это полноценный фронтенд-проект корпоративного сайта для студии интерьерного дизайна. Сайт демонстрирует услуги компании, портфолио работ, команду профессионалов и предлагает клиентам различные концепции дизайна мебели и помещений. 

Дизайн выполнен в минималистичном стиле с использованием строгой типографики, качественных изображений и приятного акцентного синего цвета.

### ✨ Ключевые особенности:
* **Адаптивный дизайн:** Полная адаптивность под мобильные устройства и планшеты.
* **Многостраничная структура:** Более 10 уникальных страниц, включая специализированные каталоги стилей.
* **Чистая архитектура:** Чистая семантическая верстка без использования тяжелых CSS-фреймворков.
* **Живой интерфейс:** Реализованы интерактивные элементы — выпадающие меню, плавные переходы и функциональные галереи.
* **SEO-оптимизация:** Использование правильной иерархии заголовков и семантических тегов для лучшей индексации поисковиками.

## 🗺️ Навигация по сайту

Проект включает в себя следующие основные разделы:
- `index.html` — Главная страница с презентацией агентства и метриками.
- `about.html` — История компании и достижения.
- `services.html` — Предоставляемые услуги.
- `portfolio.html` — Галерея выполненных проектов.
- `teams.html` — Информация о специалистах.
- `contact.html` — Форма обратной связи и контакты.

## 💻 Установка и запуск

Вы можете запустить проект локально для ознакомления или внесения изменений, выполнив следующие шаги:

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/Le9asi/dananz.git](https://github.com/Le9asi/dananz.git)
   ```

2. **Перейдите в директорию проекта:**
   ```bash
   cd dananz
   ```

3. **Запустите проект:**
   * Просто откройте файл `index.html` в любом современном браузере.
   * Или воспользуйтесь расширением **Live Server** в VS Code для автоматического обновления страницы при сохранении изменений.
